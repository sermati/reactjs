# Ecommerce Matias Serres

## Descripción:
Tienda Online - Ecommerce completo y funcional realizado para la entrega del trabajo final del curso de React JS de CoderHouse.

## ¿Que hace el sistema desarrollado?
Se pueden seleccionar productos organizados en categorias, acceder al detalle y seleccionar la cantidad necesaria para ingresar al carrito en tiempo real. Luego se puede terminar la orden de pago ingresando datos del usuario. Todo se almecena en Firebase-Firestore.

## Herramientas/Dependecias utilizadas:
* Vite 4.1.0
* React 18.2.0
* React Router DOM 6.10.0
* Bootstrap 5.2.3
* Firebase 9.20.0
* Fontawesome
* Sass 1.59.3

## Para instalar:
1. git clone https://github.com/sermati/reactjs/tree/master
2. cd ecommerce
3. npm install
4. npm start
